# AngularJS JumpStart

This repository contains the code projects and exercises for the **AngularJS JumpStart** course by Dan Wahlin on Udemy.

## About the Course

AngularJS JumpStart is a comprehensive course designed to teach the fundamentals of AngularJS (Angular 1.x), a powerful JavaScript framework for building dynamic web applications. This course covers essential concepts and provides hands-on coding exercises to help you master AngularJS development.

## Repository Structure

The repository is organized by course modules, with each module containing:

- **Begin** folder: Starting code/template files for the module exercises
- **End** folder: Completed code demonstrating the final implementation

```
angularjs-jumpstart/
└── Module1/
    └── Code/
        ├── Begin/   # Starting point for exercises
        └── End/     # Completed example code
```

## Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, or Edge)
- A code editor (VS Code, Sublime Text, Atom, etc.)
- Basic knowledge of HTML, CSS, and JavaScript

### Running the Examples

1. Clone this repository:
   ```bash
   git clone <repository-url>
   cd angularjs-jumpstart
   ```

2. Navigate to the module you want to work with:
   ```bash
   cd Module1/Code/Begin
   ```

3. Open the `index.html` file in your web browser, or use a local development server:
   ```bash
   # Using Python 3
   python -m http.server 8000
   
   # Using Node.js http-server
   npx http-server
   ```

4. Access the application in your browser at `http://localhost:8000`

## Module Overview

### Module 1: Getting Started with AngularJS

Learn the basics of AngularJS including:
- Setting up an AngularJS application
- Two-way data binding with `ng-model`
- Working with expressions and directives
- Building your first "Hello World" application

## Learning Approach

1. **Start with the Begin folder**: Each module has a Begin folder with starter code
2. **Follow along with the course**: Complete the exercises as you watch the course videos
3. **Compare with End folder**: Check your solution against the completed code in the End folder
4. **Experiment**: Modify the code and experiment with different approaches
